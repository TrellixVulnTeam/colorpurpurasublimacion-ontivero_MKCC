
El proyecto web de Color PúrPura Sublimación tiene como objetivo realizar un muestreo de productos y comunicar la lista de productos disponibles a la venta. Así como también realizar un acercamiento al usuario para realizar la compra a través de canales de contacto, enfatizando en WhatsApp. 

¿Qué encontramos en cáda página del sitio? 

Index: Muestreo de productos con precio incluido. 
Producto: Listado de productos disponibles a la venta. 
Diseños: Galería de productos reales vendidos. 
Nosotros: Información sobre el comercio e instructivo de cómo comprar.
Contacto: Formulario de contacto. 

Cada página contiene un footer donde se redirecciona a las redes sociales del comercio.
